About Me

Hello, I'm Kat and I like building things for the web. My interest in programming began when I was about 8 years old and I used my Acorn Electron to create fancy patterns and simple games. Fast forward to 1999 and we got the internet at home. I taught myself how to use Dreamweaver and then Frontpage 2000, and I've been building websites ever since.

Fast forward to today and I have just graduated (July 2022) from Code Institute's Diploma in Full Stack Software Engineering.

  • I LOVE working in tech.
  • I'm enthusiastic about adding to my tech stack (currently learning React.js).
  • I will never stop asking, "How does it all work?"

I used to work in Marketing. I was the one commissioning the websites. Now I get to build the websites! In other words, I have a strong appreciation of both sides of the table.

Portfolio

This is an ever-growing portfolio of my work. Most, but not all was completed as part of my diploma.

Watch You Want?

My final diploma project is an e-commerce website for a fictional watch shop. Provides role based permissons, authentication, email validation, CRUD and a Stripe payment system.

Lettuce Eat

A blog website based on salad. Built using Bootstrap and Django. Features an authentication mechanism and admin/user CRUD functionality.

The Sims 4 Quiz

A browser-based quiz, built in Python about Real Estate in The Sims 4. The game is timed and a user can choose to add their score to a leaderboard at the end of the game and return the leaderboard to their view.

Play Your Cards Right

This is a JavaScript game and was my second diploma project. The website is responsive on all device sizes.

Rapier Group

Great Malvern Yoga
